ORIGIN会計ソフト レシートページ
================================

目的
----
Wix上の /レシート ページで、スマホからレシート画像を撮影・保存するための準備フォルダーです。

現段階の方針
------------
固定サーバーがまだ無いため、まずはWixをスマホ入力口として使います。

初期段階では、以下を目標にします。

1. スマホでレシート撮影
2. Wixに画像アップロード
3. receipt_idを発行
4. Wix CMS「Receipts」に保存
5. 画像URL・日付・店名・金額をCSV出力できる状態にする

まだやらないこと
----------------
- OCR
- GPT
- 自動仕訳
- 本格トリミング
- PostgreSQLへの直接登録

まずは、証憑画像を安全に集める入口を作ります。

フォルダー構成
--------------
docs        説明書
wix         Wixページコード
html-frame  将来の圧縮・トリミング用HTML
local-test  ローカル確認用HTML
data        CSVサンプル
exports     CSV出力置き場
images      画像一時置き場
backup      バックアップ置き場

最初に使うファイル
------------------
wix\01_receipt_page_uploadbutton.js

これはWixの /レシート ページコードに貼るコードです。
