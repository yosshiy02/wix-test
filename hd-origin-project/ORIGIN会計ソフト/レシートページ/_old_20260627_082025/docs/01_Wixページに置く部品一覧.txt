Wix /レシート ページに置く部品一覧
==================================

まずは、Wix標準のアップロードボタンで作ります。

置く部品
--------

1. Upload Button
   ID：uploadReceiptImage
   用途：レシート画像を選択・撮影

2. Button
   ID：btnSaveReceipt
   表示：レシート保存

3. Text
   ID：txtStatus
   用途：状態表示

4. Image
   ID：imgReceiptPreview
   用途：保存後の画像プレビュー

5. Date Picker
   ID：dateReceipt
   用途：レシート日付

6. Text Input
   ID：inputVendor
   用途：店名

7. Text Input
   ID：inputTotal
   用途：合計金額

8. Text Input または Dropdown
   ID：inputPaymentMethod
   用途：支払方法

9. Text Input
   ID：inputMemo
   用途：メモ

Wix CMS
-------
コレクション名：Receipts

必要フィールドは docs\02_Wix_CMS_Receipts_項目.csv を見る。
