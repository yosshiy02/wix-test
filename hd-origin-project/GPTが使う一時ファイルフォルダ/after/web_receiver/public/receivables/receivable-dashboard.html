未消込入金
銀行入金はあるが、どの請求書に対する入金か未確定のものを表示します。
仮受金候補
相手先不明、金額不一致、用途不明の入金を仮受金候補として管理します。
未入金一覧
入金予定日を過ぎても消込されていない売掛を表示します。
督促・確認メモ
得意先への確認履歴、入金予定変更、保留理由を残せるようにします。
仕訳プレビュー 候補
入金入力から、会計用の振替伝票候補を表示します。最初から借方貸方を直接入力させず、裏で生成する方針です。
入金情報を入力して「仕訳候補作成」を押してください。
例：売掛入金なら「借方：普通預金 / 貸方：売掛金」。手数料がある場合は「支払手数料」を借方に追加します。
