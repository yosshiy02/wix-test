HD Origin Project 支払書類仕分けシステム 3段階AI外部プロンプト設計

1回目AI:
stage1-classification
目的: 共通仕分け。専門解析システムを決める。

2回目AI:
stage2-common-draft
目的: 共通下書きDB化。専門画面へ渡す候補を作る。

3回目AI:
stage3-specialist
目的: 専門解析。選ばれた専門だけ深掘りする。

重要:
- 2回目で全部解析しない。
- 3回目で専門ごとの必要項目だけ解析する。
- HTMLや画面JSでAI結果を勝手に補正しない。
- AI候補、人間選択、人間確定値を分ける。
- プロンプトは外部テキストとして管理する。
