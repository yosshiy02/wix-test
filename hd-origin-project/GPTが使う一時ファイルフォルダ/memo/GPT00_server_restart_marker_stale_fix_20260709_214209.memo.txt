==============================
GPT00 server.js 再起動マーカー stale 対策
==============================
日時: 2026/07/09 21:42:09

[症状]
通常ランチャー起動なのに、
[STARTUP_RESTORE_CHECK] 再起動ボタン経由のため、リストア確認とメイン画面OPENを省略します。
と表示されていた。

[原因]
HD_ORIGIN_RESTART_IN_PROGRESS.json の再起動マーカーが残っていると、
通常起動でも再起動ボタン経由扱いになる可能性があった。

[修正]
hdOriginConsumeRestartMarker() を修正。

- marker.reason が restart の場合だけ対象
- marker.created_at が無い場合は無視
- 5分より古い marker は無視
- 別PCの marker は無視
- 読み取った marker は必ず削除
- 古い marker は通常ランチャー起動扱いにする
