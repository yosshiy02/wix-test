==============================
GPT2 一時ファイルフォルダ 使い方
==============================

このフォルダは、GPT2作業用の一時フォルダです。

[フォルダ構造]

GPT2に渡すフォルダ
- GPT2へ見せるためのコピー置き場
- 収集前に毎回クリーンする

before
- 修正前コピー

after
- 修正版コピー

apply
- 本体へ反映するPowerShell

undo
- 元に戻すPowerShell

memo
- 確認結果・作業メモ

[禁止]
- .env は入れない
- DB は入れない
- 画像・証憑データは入れない
- backup フォルダは触らない
- 本体を直接編集せず before / after / apply / undo / memo の流れを使う

[現在の正式な収集先]
GPT2が使う一時ファイルフォルダ\GPT2に渡すフォルダ
